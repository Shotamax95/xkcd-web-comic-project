# XKCD Web Comic APP
 My App https://xkcd-web-comics.herokuapp.com/
Retrieve api from here https://xkcd.com/json.html
## Functions
* Display latest image as home.
* Next, Previous, Random button.
* Search by comic id
* Responsive Screen
